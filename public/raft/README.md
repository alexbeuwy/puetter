# Raft Images

Place your 4 hero / scroll-sequence images here as:

- `bild1.jpg` — Bretter am Ufer (Ausgangspunkt)
- `bild2.jpg` — Ins Wasser, erste Bewegung
- `bild3.jpg` — Richtung, Horizont
- `bild4.jpg` — Floß mit Motor, volle Fahrt

Recommended: ~1600px wide, JPG, warm/pastel tone that matches the
palette (sand, sky, cream). The `layout.tsx` file preloads these four
paths — if you rename them, update the `<link rel="preload">` tags.
